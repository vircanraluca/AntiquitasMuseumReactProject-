# ◈ Antiquitas — Virtual Museum

Antiquitas is a virtual museum web application that allows users to explore ancient civilisations through an interactive collection of artefacts and a historical timeline. Users can browse detailed information about legendary artefacts such as the Rosetta Stone, the Colosseum, and Machu Picchu, view a chronological timeline from Mesopotamia to the Fall of Rome, and book museum tickets directly through the platform.

## Main Features

- **Artefact Collection** — Browse 6 legendary artefacts with detailed descriptions and historical facts
- **Timeline** — Explore key moments in ancient history presented in an animated timeline
- **Ticket Booking** — Select ticket types, choose a visit date, and confirm bookings saved to the database
- **Cart** — Add multiple ticket types to a cart and review the order before confirming

## Technologies

React Router v7, TypeScript, Tailwind CSS, Node.js, Express.js, MySQL
